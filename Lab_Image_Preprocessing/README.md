# Affine Normalization in Data Preprocessing

This project explores data normalization from a geometric perspective, showing how several common normalization techniques in machine learning can be understood as affine transformations.

## 📁 Project Structure

- `notebook/`
  - `Affine_Normalization_Geometry.ipynb`  
    Conceptual explanation of affine transformations and their effect on data distributions.
  
  - `Affine_Normalization_Application.ipynb`  
    Practical examples of normalization techniques used in machine learning, including Min-Max scaling, Z-score normalization, and image preprocessing. I have applied these to the class "dandelion" of the Flowers dataset.

## 🧠 Key Ideas

- Affine transformations modify data through scaling and translation.
- They change location and dispersion but preserve the overall shape of the distribution.
- Many normalization techniques used in machine learning are affine transformations.
- Normalization in image preprocessing can be understood as an affine transformation.

## 🚀 Motivation

Normalization is a fundamental step in data preprocessing, yet it can be confusing due to the variety of techniques available.  
This project provides a unified perspective better to understand these methods and their impact on data.
